# Code Of Conduct

Contributors are expected to keep collaboration direct, respectful, and
constructive.

## Expected Behavior

- Critique code and decisions, not people.
- Share concrete evidence when raising concerns.
- Respect privacy and security boundaries.
- Help keep the repository safe for public collaboration.

## Unacceptable Behavior

- Harassment or personal attacks
- Publishing private or identifying information without consent
- Deliberately disruptive behavior
- Knowingly introducing malicious or unsafe changes

Maintainers may moderate discussions and contributions to protect the project
and its contributors.
